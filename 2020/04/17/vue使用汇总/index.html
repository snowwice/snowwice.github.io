视图更新解决方法

视图更新问题遇到了太多太多次，主要用以下的解决方法：

  1. this.$forceUpdate()
    使用场景：数据层级太多造成的更新不及时，一般还可以用this.$set(目标对象,索引,值)或js的操作对象和数组的方法代替，但貌似这个更好用。
  2. this.$nextTick(() => {})
    使用场景：dom生成之后的视图更新
  3. v-if展示标识
    使用场景：一些诡异的视图不更新，一般这个解决不了就看看自己有没有犯一些弱智的错误吧。
  4. key（组件更新）
    使用场景：v-if加载的组件不更新，可能是相似组件不会更新元素。一般很少使用，检查+1。
